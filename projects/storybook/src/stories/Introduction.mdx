# Bienvenido a la Documentación

Bienvenido a la documentación de Storybook para nuestro proyecto Angular. Aquí podrás explorar y probar todos los componentes de nuestra aplicación de manera interactiva. Storybook es una herramienta poderosa que te permite desarrollar y documentar componentes de forma aislada, lo que facilita la creación de interfaces de usuario coherentes y reutilizables.

## ¿Qué es Storybook?

Storybook es un entorno de desarrollo para componentes de interfaz de usuario. Permite a los desarrolladores crear y probar componentes en un entorno aislado, lo que facilita la detección de errores y la mejora de la calidad del código. Con Storybook, puedes ver y interactuar con los componentes en diferentes estados y configuraciones, lo que simplifica el proceso de desarrollo y prueba.

## Estructura del Proyecto

En este proyecto, hemos configurado Storybook para trabajar con Angular. La estructura del proyecto incluye:

- **Componentes**: Todos los componentes reutilizables de la aplicación están documentados aquí. Cada componente tiene su propia historia que muestra sus diferentes estados y variaciones.
- **Documentación**: Puedes encontrar documentación sobre cómo utilizar cada componente, así como guías y mejores prácticas para desarrolladores.
- **Interactuar**: Usa el panel de controles para ajustar las propiedades de los componentes y ver cómo cambian en tiempo real.

## Cómo Empezar

Para comenzar a explorar los componentes, navega a la sección de "Componentes" en el menú de Storybook. Aquí encontrarás una lista de todos los componentes disponibles. Cada historia incluye una demostración en vivo y una descripción detallada del componente.

Si eres nuevo en el proyecto o en Storybook, te recomendamos comenzar con la sección de introducción y familiarizarte con la estructura y los componentes principales. Si tienes alguna pregunta o encuentras algún problema, no dudes en consultar la documentación o contactar al equipo de desarrollo.

¡Disfruta explorando y desarrollando con Storybook!
